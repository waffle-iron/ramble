# ramble

# Front End (MVP)
* Login / Signup -- `/signup` & `/signin`
* View Entries -- `/dashboard`
* View Single Entry -- `/post/:id`
* Create / Edit / Destroy Entries -- `/post/:id`

* Random Prompt System
* Delay Delete / Scheduled Delete
* Markdown Support
